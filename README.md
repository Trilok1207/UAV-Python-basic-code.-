# UAV Autonomous Navigation using SLAM, A* and PID

## 🚀 Overview
This project implements an autonomous UAV navigation system using:
- SLAM (Occupancy Grid Mapping)
- A* Path Planning
- Dynamic Obstacle Avoidance
- PID Control for trajectory tracking

## 🧠 Features
- Real-time map building (simulated SLAM)
- Multi-obstacle environment
- Dynamic obstacle handling
- Path replanning using A*
- Smooth trajectory tracking using PID

## ⚙️ Technologies Used
- MATLAB
- Control Systems (PID)
- Path Planning Algorithms (A*)

## 📊 Output
The UAV successfully navigates from start to goal while avoiding static and dynamic obstacles.

## 📌 How to Run
1. Open MATLAB
2. Run `main.m`
3. Observe trajectory and map visualization
